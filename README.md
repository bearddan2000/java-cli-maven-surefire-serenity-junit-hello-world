# java-cli-maven-surefire-serenity-junit-hello-world

## Description
A POC for maven app using junit5
serenity framework
 with surefire plugin.

## Tech stack
- java
- maven
  - junit5
  - surefire
  - cucumber
	- serenity



## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
